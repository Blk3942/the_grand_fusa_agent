# 4. Item 概述

- Item 名称：驾驶员监控系统（Driver Monitoring System, DMS）
- 功能目标：在车辆运行过程中监测驾驶员注意力与状态（分心、疲劳、闭眼、视线偏离等），并在满足触发条件时向驾驶员和上层功能提供告警/状态输出。
- 非安全目标（如性能/舒适性）：降低误报率、提升告警可接受性、优化夜间与逆光场景可用性。

# 5. 边界与接口

## 5.1 系统边界

- 在范围（In Scope）：
  - 驾驶员舱内图像采集（红外/可见光摄像头，具体型号待定）
  - 驾驶员状态估计与分类算法
  - DMS 状态机与告警触发逻辑
  - 向 HMI（Human Machine Interface，人机交互）与车辆域控制器输出状态信号
- 不在范围（Out of Scope）：
  - 自动驾驶策略决策本体
  - 制动/转向执行机构控制
  - 驾驶员身份识别与娱乐功能

| IF-ID | 接口对象 | 输入/输出 | 信号/消息 | 正常范围 | 异常处理 |

# 6. 运行场景与运行条件

| SCN-ID | 场景描述 | 速度/道路/天气等条件 | 驾驶员状态 | 备注 |
|---|---|---|---|---|
| SCN-DMS-001 | 城市跟车场景下注意力监控 | 0-80 km/h，白天/夜晚，城市道路 | 正常驾驶 | 基线场景 |
| SCN-DMS-002 | 高速巡航场景下疲劳监控 | 60-120 km/h，高速道路 | 疲劳累积、眨眼频繁 | 重点安全场景 |
| SCN-DMS-003 | 夜间逆光/遮挡场景 | 夜间、隧道出入口、镜片反光 | 头部偏转或部分遮挡 | 感知鲁棒性边界场景 |
| SCN-DMS-004 | 低速拥堵与短时视线转移 | 0-30 km/h，拥堵路况 | 临时视线偏离 | 需区分正常行为与风险行为 |
